Codex AI, a prominent desktop application leveraging artificial intelligence for development tasks, has rolled out a significant new feature named "Conexiones" (Connections). This innovation enables users to remotely control their desktop-installed Codex instance directly from a mobile device. The core functionality allows instructions to be dispatched from anywhere, empowering Codex to modify websites, execute console programs, or implement code changes on the host computer without requiring direct keyboard input. This capability substantially enhances productivity and flexibility, positioning "Conexiones" as a critical addition for developers seeking streamlined, remote management of their development environments.

Establishing a connection is a straightforward process, typically initiated by scanning a QR code from the ChatGPT mobile application's dedicated Codex tab. Once linked, the mobile device serves as an effective messaging interface, facilitating ongoing conversations, project execution, file updates (such as READMEs), and dynamic permission management, including granting full command execution access. The mobile interface provides real-time updates, displaying summaries of command executions and screenshots captured by invoked skills, like Playwright CLI for responsive design testing. Furthermore, integration with tools such as Cloudflare can expose local development projects via temporary public URLs, enabling remote testing and review directly from the mobile device. The desktop application's "Conexiones" settings also offer robust management options, including the ability to prevent the machine from entering sleep mode during remote operations and the flexibility to connect multiple devices simultaneously, encompassing additional mobile phones or even a secondary laptop, thereby ensuring continuous and versatile control over the primary Codex instance.
