A recent, severe supply chain attack has targeted the JavaScript ecosystem, compromising over 80 malicious versions across 42 packages within the popular Tstack family, including React Query and Tstack Star. The attack, exploiting GitHub Actions cache poisoning rather than direct NPM token theft, is highly destructive, stealing critical environment variables (AWS, Google Cloud, GitHub, SSH) and installing a 'death switch' that can wipe the entire hard drive if exposed tokens are revoked. While primarily affecting JavaScript, similar incidents have also been observed in Python and Ruby ecosystems. Developers are strongly advised to transition from default `npm install` to `pnpm` version 11 or higher, which offers built-in supply chain protection through features like `minimum-release-age` and `block-exotic-sub-deps`, and by default avoids executing `postinstall` and `preinstall` scripts from sub-dependencies, thereby mitigating significant attack vectors.

Concurrently, the development of the official 'La Velada del Año 6' website (infolavelada.com) is progressing as an ambitious open-source initiative. This project leverages AI coding assistants such as Cursor (utilizing Opus 4.7 and GPT 5.5) and Codex for iterative design and implementation of key sections, including sponsor displays, an interactive fighter selection interface, and various video content integrations. A significant job offering, sponsored by Infojobs, attracted over 30,000 applicants, with four developers slated for hire. The live development session underscored both the efficiency and challenges of AI in generating code and design, often requiring precise prompt engineering and manual refinement. Further job opportunities, including the development of a website for a world champion footballer, are expected soon. The session also confirmed a live broadcast of the Google I/O keynote next Tuesday.
