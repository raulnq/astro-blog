Google has officially released Antigravity 2.0, a significant update that redefines the platform by pivoting from its origins as a code editor to an AI-powered desktop application. This strategic shift positions Antigravity 2.0 as a direct competitor to tools like OpenAI's GPT Codex and Cloud Code Desktop, emphasizing multi-session, chat-driven code generation and project management without the need for a traditional IDE. The Antigravity brand now encompasses a broader suite, including the new 2.0 desktop app, a command-line interface (Antigravity CLI), and a Software Development Kit (Antigravity SDK) for agent creation. Initial community reception has been largely negative, with many existing users expressing dissatisfaction over bugs encountered during download, the loss of previous editor configurations and extensions, and the fundamental departure from Antigravity's initial identity.

The new Antigravity 2.0 desktop application provides an interface for rapid project creation, allowing users to prompt the AI to build applications, generate code, and manage development workflows. Key features include an interactive chat environment for issuing commands, project-specific configuration options, security presets for command execution, and a plan overview for AI-driven tasks. It supports AI models such as Gemini 3.5 Flash, though free-tier usage is subject to rapid consumption limits, necessitating a subscription for sustained use. While it offers advanced functionalities like goal-driven execution, browser invocation for project testing, and code review with integrated diff views for suggested modifications, the platform currently lacks an integrated terminal and browser found in some competitors. Additionally, the desktop app does not feature an in-app marketplace for "skills" or extensions, relying on external configurations or the CLI. Despite its resemblance to other AI-centric development platforms, Antigravity 2.0 is seen by many as being less mature, particularly when compared to its more robust command-line counterpart, Antigravity CLI, or established rivals like Codex. Google's direction appears to cater to a "by coder" user base, prioritizing prompt-based interaction over traditional developer workflows, which has alienated a segment of its original user base.
