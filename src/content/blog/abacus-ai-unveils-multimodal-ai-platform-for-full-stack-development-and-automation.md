Abacus AI, previously referenced as App Lelm, has unveiled a comprehensive, browser-based, AI-driven platform designed to streamline full-stack application development. The platform empowers users to generate complex, multi-platform applications—encompassing web frontends, robust backends, and native mobile components—entirely through natural language chat prompts. Leveraging multi-agent AI execution via its "EN Swarm" feature, Abacus AI accelerates code generation, providing integrated database management and project hosting. Demonstrations showcased the rapid creation of a personal finance tracker with synchronized web (built with Next.js) and mobile (built with React Native) applications, as well as a sophisticated CRM featuring client tracking, user roles, and integrated Stripe payment processing. Abacus AI delivers production-ready, extensible solutions, offering downloadable source code and comprehensive API documentation via Swagger.

Beyond core application generation, Abacus AI extends its utility with a powerful suite of AI and automation capabilities. Users can interact with existing Google Drive documents or Excel sheets through chat for data analysis and summarization, generate presentations by fetching information and images from the internet, and implement browser automation for scheduled data extraction from various websites. The platform also integrates "Supercuter," enabling the execution of cloud-based AI agents (including Cloud Code and GPT models) without reliance on external API calls, ensuring local processing. Furthermore, Abacus AI offers a versatile multi-model chat interface supporting a range of AI models such as Cloud, GPT, Kimi, and open-source options, alongside specialized tools like Nanobanana for image generation. An intelligent "Rout the LM" function automatically selects the most appropriate AI model for given tasks, positioning Abacus AI as an all-encompassing environment for diverse software development, content creation, and workflow automation needs.
