Cubath has announced a new hackathon, inviting developers to create AI-driven applications hosted on their VPS infrastructure, with over $1,300 in prizes. Running from March 18th to March 31st (23:59:59 CET), participants must deploy new, open-source projects on Cubath and register via a GitHub Issue. Cubath, the event's sponsor, offers $15 in free credit without requiring a credit card, emphasizing its low-latency data centers, including one in Barcelona, which is particularly beneficial for European users. The hackathon's accompanying technical demonstration showcased building a server-side event (SSE) AI API from scratch using Bun, a fast JavaScript runtime, and integrating multiple free AI models like Open Router's Free Models Router, Grock, and Cerebras (specifically using the Llama model) to mitigate rate limits through a round-robin strategy.

The deployment process was detailed using Docploy, a self-hosted alternative to platforms like Vercel, running on a Cubath VPS. The demonstration covered seamless GitHub integration for auto-deployment on push, the use of Railway Buildpack for automated project builds, custom domain configuration with HTTPS via Let's Encrypt, and secure management of environment variables. A PostgreSQL database was also deployed within Docploy, ensuring zero-latency internal connectivity to the API. For workflow automation, an N8N instance was deployed and configured to trigger Slack notifications via a Docploy webhook upon deployment or build errors. This notification leverages OpenAI to generate context-aware, humorous messages. This comprehensive setup highlighted Cubath's integrated ecosystem, enabling developers to deploy various services—from web applications and APIs to databases and automation tools—with single-click installations, even on entry-level "GP Nano" VPS instances.
