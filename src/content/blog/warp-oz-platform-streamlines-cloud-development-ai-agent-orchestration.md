Warp, known for its terminal innovations, has officially unveiled Oz, a new orchestration platform poised to redefine how developers manage cloud-based workflows through AI agents. Positioned as a catalyst for 'agent orchestration' in 2026, following 2025's focus on 'interactive agents,' Oz allows for the execution of agents directly in the cloud, ensuring development continuity irrespective of local machine status. This platform addresses the growing challenge of staying abreast of constant updates in cloud environments like AWS, enabling autonomous operations by transforming agent skills into automated tasks and integrating agents into applications to manage repetitive development processes. Oz facilitates the parallel execution of an unlimited number of cloud coding agents, significantly offloading computational demands from local machines.

The power of Oz was recently demonstrated through the construction of an automated AWS cloud news dashboard. Starting with a GitHub repository, developers can leverage an AI agent via the Oz platform (accessible through a web dashboard or Warp terminal) to generate an entire base project, including fetching and displaying AWS news from various sources. The demonstration highlighted Oz's capacity for parallel processing, running tasks such as a breaking news ticker and a code security scan concurrently in the cloud, while a local agent built the dashboard. Post-creation, a GitHub Actions workflow deployed the dashboard, and an AI scheduler was configured to automatically update the news daily, ensuring the dashboard remains current without manual intervention. Furthermore, Oz introduces real-time agent session sharing, fostering collaborative development by allowing multiple team members to observe and steer an agent's execution, instantly adapting its focus based on collaborative input. This comprehensive approach to agent-driven development and automation underscores Oz's potential as a transformative tool for modern cloud practitioners.
